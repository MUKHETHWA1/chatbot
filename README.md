## Cybersecurity Awareness Chatbot

A desktop chatbot application built with WPF (.NET Framework/Core) that helps users learn about cybersecurity best practices through interactive conversations. The bot features voice greetings, sentiment detection, memory recall, and dynamic responses on topics like passwords, scams, privacy, and phishing.

## Core Functionality
- Voice Greeting - Plays a recorded WAV greeting when the application starts
- Personalized Responses - Remembers and uses the user's name throughout the conversation
- Keyword Recognition - Detects cybersecurity topics and provides relevant responses
- Random Responses - Multiple pre-defined responses for variety in conversations
- Conversation Flow - Handles follow-up questions like "tell me more" or "another tip"
- Sentiment Detection - Detects user mood (worried, frustrated, curious, happy)
